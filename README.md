# frp 安卓客户端

[中文文档](README.md) | [Readme](README_en.md)

由 frp v0.37.1编译得到
目前只有frpc, frps还没有实现
支持各种协议 http, https, tcp, udp, xtcp, xudp
测试在android 11可以运行, 其它没测试过

## 使用

填写frpc.ini文件, 点击start启动frpc服务, 点击stop停止frpc服务
